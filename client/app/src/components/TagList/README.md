## TagList Component
A reusable tag list for blog articles.

### Example

```js
<TagList tags={tags} />
```

### Props

| Prop          | Type     | Default     | Possible Values
| ------------- | -------- | ----------- | ---------------------------------------------
| **Separator**    | String   |  ', '           | Any string value. Used to separate each tag.
| **tags**    | An array of tags   |             | An array of objects with a title value.
